# Data Structures and Algorithms in C

Implementations of fundamental data structures, classic algorithms, and LeetCode solutions written in C.

This repository was developed during the **Data Structures and Algorithms** K-MOOC course completed alongside my BSc studies in Computer Engineering at **Óbuda University**. The entire learning process was tracked with Git, making this repository a complete record of the course from start to finish.

## Contents

### Data Structures
- Arrays & Matrices
- Singly & Doubly Linked Lists
- Stacks & Queues
- Binary Trees & Binary Search Trees
- Static & Dynamic Hash Tables

### Algorithms
- Linear & Binary Search
- Insertion, Merge, Quick, Heap, Counting & Radix Sort
- Huffman Coding

### LeetCode
- 25+ problems solved in C
- Original task descriptions included
- Focus on clean implementations and algorithmic efficiency

## Project Highlights

- Modular project structure (`include/` / `src/`)
- Manual memory management
- Pointer-based implementations
- CMake build system
- Emphasis on clean code and Big-O complexity analysis
